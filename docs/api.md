# API

- `/api/slates` (`GET, POST`)
- `/api/proposals` (`GET, POST`)
- `/api/ballots` (`POST`)

## Endpoints

### `GET slates`
Get a list of all slates.

### `POST slates` 
Save information for a slate.

### `GET proposals`
Get all proposals.

### `POST proposals`
Create a new proposal.

### `POST ballots`
Save information for a given user's ballot.
